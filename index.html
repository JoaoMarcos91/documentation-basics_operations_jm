Introdução

Bem-vindo à documentação da biblioteca basicsoperations Python

Instalação

Você deve rodar esse comando no seu terminal para poder realizar a instalação do basicsoperations.

pip install basicsoperations

Importação

A importação da biblioteca 

basicsoperations
é bastante simples, basta adicionar essa linha de código no início de seu código Python:

import basicsoperations

Exemplo de uso: 

import basicsoperations


result_soma = basicsoperations.soma(2, 3, 4)

result_subtracao = basicsoperations.subtracao(5, 2, 4)

result_multi =  basicsoperations.multiplicacao(4, 2, 0.5)

result_divisao = basicsoperations.divisao(10, 2, 3)

Referência da API

A API da biblioteca basicsoperations na versão mais recente (0.1.0) suporta as seguintes funções:

  • soma(*args): Soma todos os números colocados dentro dos parênteses.
  • subtracao(*args): Subtrai todos os números colocados dentro dos parênteses em ordem da esquerda para a direita.
  • multiplicacao(*args): Multiplica todos os números colocados dentro dos parênteses.
  • dividisao(*args): Divide todos os números colocados dentro dos parênteses em ordem da esquerda para a direita.

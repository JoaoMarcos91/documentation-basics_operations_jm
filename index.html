Introdução

Bem-vindo à documentação da minha biblioteca Python!

Instalação

pip install minha-biblioteca

Importação

A biblioteca 

basics_operations_jm
comporta algumas funções já prontas, as quais você pode importar separadamente. Aqui estão elas:

Para importar a biblioteca inteira

from basics_operations_jm import *

Exemplo de uso: 

from basics_operations_jm import *


result_soma = basics_operations_jm.soma(2, 3, 4)

result_subtracao = basics_operations_jm.subtracao(5, 2)

result_multi =  basics_operations_jm.multi(4, 2)

result_divisao = basics_operations_jm.divisao(10, 2)

Para importar funções separadamente

from basics_operations_jm import <nome-da-função>

Exemplos

Aqui estão alguns exemplos de uso:


import minha_biblioteca
print(minha_biblioteca.somar(1, 2, 3))

Referência da API

A API da biblioteca suporta as seguintes funções:

  • somar(*args): Soma números.
  • multiplicar(*args): Multiplica números.
  • dividir(*args): Divide números.
